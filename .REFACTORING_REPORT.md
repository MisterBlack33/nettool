## 3. AUTOMATISIERBARE REFACTORINGS

### Catch-Block-Template (für PowerShell/Sed):
```regex
SEARCH: catch\s*\(\s*(\w+)\s+(\w+)\s*\)\s*\{\s*\}
REPLACE: catch ($1 $2) {\n    LOGGER.log(Level.FINE, "Exception in $CONTEXT", $2);\n}
```

### Utility-Umbenennung:
```bash
find src -name "*Utils.java" -o -name "*Helper.java" | xargs sed -i 's/class \(.*\)Utils/class \1Accessor/g'
```
